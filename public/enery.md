## Working Style

- 优先保持现有业务逻辑不变
- 优先保持原有 CSS Class 不变
- 修改代码时尽量最小化变更范围
- 先分析原因，再给出修改方案
- 输出可直接复制使用的代码
- 涉及 Vue 项目时优先考虑 Element UI / Element Plus 生态
- 涉及数据库时优先考虑 MyBatis 实现
- 遇到错误时先定位根因，不要直接重构
